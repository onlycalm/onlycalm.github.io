GitKraken破解

前言：
  GitKraken是一款非常好用的Git可视化管理工具，由于该软件是收费软件，因此本篇文章介绍一种破解方法。

1 GitKraken简介

   GitKraken 是一款界面十分美观的Git可视化管理工具，git的指令操作都变为点击鼠标完成，虽然我是git指令用户，但是借助GitKraken可以使我非常直观的浏览版本提交的各分支之间的关系，以及对任意两次提交进行差异对比，甚至提前观察到合并后的结果。比较遗憾的是GitKraken是收费软件，类似的免费软件有 Sourcetree ，不过使用体验不如GitKraken，因此本篇文章介绍GitKraken的破解方法。

2 下载安装GitKraken

2.1 下载

   登录GitKraken官网找到自己操作系统对应的版本下载安装，官网上提供了Windows、Mac、Linux三个平台的版本，本篇文章以Windows（64-bit）版本介绍。
  细心点可以发现，官网上下载的原版本支持7天试用。我们先下载一个最新版安装。

2.2 安装

  安装过程是全自动的，双击 GitKrakenSetup.exe 开始自动安装，会直接安装在默认路径中。
  软件安装完成后会出现登录界面，注册登录即可，或者选择第三方账号登录，建议使用GitHub账号登录。我使用GitHub登录因此直接点击 Sign in with GitHub 。

  之后会弹出网页，在网页上登录GitHub账号，如果没有将GitKraken和GitHub绑定过，还需要在网页上根据提示绑定即可。和GitHub绑定成功后GitHub的 Applications 中可以看到增加了GItKraken图标。

  在回到GItKraken界面，此时可以看到要求设置头像和用户名的界面，填写完成后点击 Save changes 即可。

  之后会弹出界面提示你选择购买正版或试用7天，由于我已经体验了6天，因此这里显示还剩1天，但如果是新用户有7天体验时间。

  在底部的状态栏可以清楚看到剩余的天眼天数，并且版本为 TRIAL 试用版。

  如果七天使用时间到期后，右下角的 TRIAL 就会变为 FREE ，这意味着GitKraken只允许你免费用于开源仓库，私有仓库必须充值。

3 破解

  破解需要确保GitKraken已登录账号并且当前正处于21天试用期，否则破解可能不成功。
  当前最新版本为v7.7.2，本篇破解也是基于该版本，若将来此方法无法破解最新版本，可在 我的百度云盘中下载 v7.7.2 （提取码：8yoj）。
  破解前先确保GitKraken已关闭退出，破解工具在Github上是一个开源项目 5cr1pt/GitCracken ，将该项目clone到本地，使用方法也可参考项目的README.md文件，另外也在Gitee上也找到了。本篇文章所使用的版本也上传在云盘中，若要使用此版本（Commit ID: 192c695 ）可在上面云盘中找到。
  接下来要用到yarn指令，因此需要确保已经安装了Yarn。

3.1 安装Yarn

  如果已经安装了npm，则通过npm执行以下指令安装Yarn。

  安装成功后输入指令 yarn -v 显示版本号则说明安装成功。

  如果没有安装npm，则先进行npm安装。
  安装npm需要安装nodejs，因为npm是其下的包管理器，登录 nodejs 官网下载长期维护的稳定版，双击msi文件进行安装，按照默认一直点下一步即可。
  安装完成后在终端命令行中输入 npm -v ，若能看到版本号则说明npm安装完成。

3.2 运行破解项目

  仓库拷贝到本地后，cd进入仓库的 GitCracken/GitCracken 目录，依次执行以下指令运行破解。

  • yarn install
  • yarn build
  • node dist/bin/gitcracken.js patcher
