英语基础语法（四）——形容词

前言：
  形容词是最重要的实词之一，在句中起重要的作用。形容词有比较级和最高级d额变化。

1 什么是形容词

  形容词用于表示人或事物的性质或特征，用于修饰名词或代词的词称为形容词。翻译为汉语“…的”。

2 形容词的句法功能

  形容词在句中一般充当以下成分：定语、宾语、表语、宾语补足语、状语和独立成分。

3 形容词的构词

3.1 名词/动词 + 后缀

3.1.1 名词 + 后缀

  • 名词 + -ful
  • 名词 + -less
  • 名词 + -en
  • 名词 + -ous
  • 名词 + -y
  • 名词 + -ish
  • 名词 + -ible
  • 名词 + -ary
  • 名词 + -al

3.1.2 动词 + 后缀

  • 动词 + -able
  • 动词 + -ive
  • 动词 + -ing
  • 动词 + -ed

3.2 前缀 + 形容词

  • un-
  • dis-
  • im-
  • in-
  • il-
  • ir-

3.3 复合形容词

  复合形容词通常由两个或两个以上的单词组成，在句中通常做定语，单词之间要用连字符。

  • 形容词 + 名词（+ 其他）：
    • 形容词 + 名词
    • 形容词 + 名词 + -ed
  • 形容词/副词 + 形容词
  • 名词 + 形容词
  • 名词 + 分词：
    • 名词 + 现在分词
    • 名词 + 过去分词
  • 数词 + 名词（+ 其他）
    • 数词 + 名词（单数形式）
    • 数词 + 名词（单数形式） + 形容词
    • 数词 + 名词（单数形式） + -ed

4 形容词的分类

  • 描述性形容词：描述名词或代词的性质、状态、大小、颜色等。
  • 数量形容词：
    • 数词。
    • 表示倍数的词。
    • 表示不定数量的形容词（some, any）。
  • 指示形容词（this, that, these, those）。
  • 物主形容词。
  • 名词所有格。
  • 疑问形容词（what, which, whose）。
  • 专有形容词。
  • 动词形容词（动词的-ing/-ed形式）。

5 形容词比较级等级

5.1 比较级

  • 一般情况下，在形容词词尾直接加-er。
  • 以-e结尾的形容词，在词尾加-r。
  • 以重读闭音节 + 辅音字母结尾的形容词，双写词尾辅音字母再加-er。
  • 以辅音字母加-y结尾的形容词，变-y为i再加-er。
  • 双音节或多音节形容词应在其前面加单词more。
  • less也可以修饰比较级，与more意义相反，意为“较少的“。
  • 形容词比较级的不规则变化。

5.2 最高级

  • 一般情况下，在形容词词尾直接加-est。
  • 以不发音的字母-e结尾的形容词，在词尾加-st。
  • 以重读闭音节 + 辅音字母结尾的形容词，双写词尾辅音字母再加-est。
  • 以辅音字母加-y结尾的形容词，变-y为i再加-est。
  • 双音节或多音节形容词应在其前面加单词the most。
  • the least也可以修饰最高级，与the most意义相反，意为“最少的”。
  • 形容词最高级的不规则变化。

6 形容词比较级的用法

6.1 原级间的比较

  • 表示两者相比（A = B）时，句型为：主语（A） + 谓语 + as + 形容词原级 + （所修饰的名词+） as + 比较对象（B）。
  • 表示两者相比（A ≠ B）时，句型为：主语（A） + 谓语 + not so（as） + 形容词原级 + （所修饰的名词+） as + 比较对象（B）。
  • 表示倍数时，句型为：主语（B） + 谓语 + …times as + 形容词原级 + as + 比较对象（B）。
  • 表示“一半“时，句型为：主语（A） + 谓语 + half as + 形容词原级 + as + 比较对象（B）。
  • 为了避免重复，通常后面的比较对象用one或ones来代替。

6.2 形容词比较级的基本句型

  • 形容词比较级主要用于两者之间的相互比较（A > B或A < B），句型为：主语（A） + 谓语（系动词） + 形容词比较级 + （所修饰的名词）than + 比较对象（B）。
  • 在形容词比较级前可用much, even, still, a lot, far, any等词来修饰。
  • 有时形容词比较级后面不用than，所比较的对方（B）省略。句型为：主语（A） + 谓语（系统词） + 形容词比较级。
  • 两者间比较（A > B或A < B），也可用“the + 比较级 + of短语（比较范围）”的形式来表示。
  • 两者比较，所比较的内容要一致。
  • 某人或某物与几个同类人或事物比较时，应用“than + any other + 名词单数或than + any of the other + 名词复数“形式。
  • ”比较级 + and + 比较级“，表示”越来越…“。如果是双音节或多音节形容词，则为“more and more + 形容词”。
  • “The + 比较级（+ 主语 + 谓语），the + 比较级（+主语 + 谓语）”，表示“越…，越…”。
  • 使用形容词比较级时，应注意以下几点：
    • 如果在同一范围内比较，进行比较的双方互不包含。此时应该用“any other + 名词单数”，或anyone else“。
    • 比较的对象是同类或对等关系时。
    • 注意区别以下词组的含义。
词组 含义
more than 多于
no more than 只不过，只有
not more than 不超过
less than 少于，不到
no less than 不亚于，有…之多
not less than 不少于

  no + 形容词比较级 + than表示“和…不一样”，not + 形容词比较级 + than表示“不及…”。

6.3 形容词最高级的基本句型

  • 形容词的最高级主要用于三者或三者以上的人或物的比较，其句型为：主语 + 谓语（系动词） + the + 形容词最高级 + （所修饰的名词） + ）比较的范围（短语或从句）。
  • 有时形容词最高级后面不加比较范围。句型为：主语（A） + 谓语（系动词） + 形容词最高级。
  • 使用形容词最高级，应注意以下几点：
    • 形容词最高级前面一般要加定冠词，但当其用做表语而又不是同其他人或事物相比较时，定冠词常可省略。
    • 形容词最高级前有物主代词时，冠词需省略。
    • 在“most + 形容词 + 名词”的结构中，此时most是程度副词，表示“极、很、非常、十分”等意思，不表示最高级。
    • 三者或三者以上相比，有时可以用“比较级 + （它所修饰的名词 +）than + 比较对象”的形式，其含义是“A比其他都…”之意。
    • 为了避免重复，后面的比较对象通常用one(s)来代替。

7 形容词的句法功能

  • 做定语。
  • 做表语。
  • 做宾语补足语。
  • 做状语。
  • 做独立成分。

8 形容词的位置

8.1 做定语时

8.1.1 置于名词前

  形容词作定语时，通常放在名词前面。如果还有其他词修饰该名词，则需要考虑其与被修饰名词之间的密切程度。通常情况下，与名词联系越紧密的形容词就越靠近名词。

  • “限定词 + 形容词 + 名词”。如果限定词（冠词/指示代词/人称代词/名词所有格）与形容词同时出现，通常把限定词放在第一位。
  • “大小 + 新旧 + 颜色 + 类别 + 名词“。如果有表示大小、新旧、颜色、类别的形容词，则按照顺序排列形容词。
  • “序数词 + 基数词 + 名词”。如果序数词（以及next, last）与基数词同时出现，则序数词（以及能next, last）放在基数词之前。
  • 如果有两个或两个以上的形容词修饰名词，与被修饰的名词关系较紧密的形容词靠近该名词。
  • 如果形容词与名词的紧密程度一样，则音节较短的形容词放在音节较长的形容词前面。

8.1.2 置于名词/代词后

  • ”复合不定代词 + 形容词“。此类复合不定代词有something, anything, everything, nothing, someone, anyone, everyone, no one, somebody, anybody, everybody, nobody等。
  • ”名词，+ 形容词A + and/or + 形容词B“。由and/or连接的成对的形容词做定语，为了句子的节奏与平衡，通常放在被修饰的名词之后。
  • “名词 + 形容词短语”。如果是形容词短语做定语，为了避免头重脚轻，形容词短语要放在名词之后。

8.2 做表语时

  形容词做表语通常置于动词之后：“主语 + 谓语动词 + 形容词”。

8.3 做宾语补足语时

  形容词做宾语补足语：“动词 + 宾语 + 形容词”。

9 形容词与其他词类的互换

9.1 名词

  • of + 抽象名词 = 形容词。of + 抽象名词相当于形容词的作用，常见的词类名词有importance, significance, help, use, value, difference等，其前通常可以用great, much, some, any, little, no等词修饰。
  • the + 形容词 = 名词。
    • 表示一类人或物。
      • 指代具体名词，表示复数概念。
      • 指代抽象名词，表示单数概念。
    • 表示单个概念，具有某种性质的个体。
    • 表示一部分。

9.2 动词

  be动词 + 形容词 + 介词 = 及物动词。

9.3 副词

  • 介词 + 形容词 = 副词。
  • to be + 形容词 = 副词。
  • 前面的形容词加重后面所修饰形容词的程度，与原词义无关，起副词的作用。

9.4 形容词短语

  • 形容词（A） + 并列连词 + 形容词（B）。
  • 副词 + 形容词。通常情况下，此类副词是表示程度的副词。
  • 形容词 + 动词不定式。
