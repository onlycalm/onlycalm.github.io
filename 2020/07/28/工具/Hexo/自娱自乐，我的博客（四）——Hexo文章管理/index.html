Front-matter

  我们写的文章需要放在Hexo根目录下的source/_posts目录下，可根据专题分类创建文件夹分类放，但是如何在网页上实现文章分类并给文章打上标签呢。这时候需要用到 Front-matter 配置。它是放在md文件开头能被Hexo解析识别的一种语法，写在 --- 之间，常用参数有如下几种。

参数 默认值 描述
layout - 布局
title 文件名 标题
date 文件创建时间 创建时间
updated 文件更新日期 更新时间
comments true 开启文章的评论功能
tags - 标签（不适用于分页）
categories - 分类（不适用于分页）
permalink - 覆盖文章地址
