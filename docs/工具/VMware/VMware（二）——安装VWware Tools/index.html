VMware（二）——安装VWware Tools

前言：
&emsp;&emsp;为了更好的在实体机中使用虚拟机，VMware Tools是必不可少的工具，本篇文章带你安装使用它。

1 功能介绍

&emsp;&emsp;VMware Tools是一套可以提高虚拟机客户机操作系统性能并改善虚拟机管理的实用工具。主要功能如下：

  • 支持 Aero 的操作系统上大大提升的图形性能和 Windows Aero 体验；
  • 使虚拟机中的应用程序像其他任何应用程序窗口一样显示在主机桌面上的 Unity 功能；
  • 主机与客户机文件系统之间的共享文件夹；
  • 在虚拟机与主机或客户端桌面之间复制并粘贴文本、图形和文件；
  • 改进的鼠标性能；
  • 虚拟机中的时钟与主机或客户端桌面上的时钟同步；
  • 帮助自动执行客户机操作系统操作的脚本；
  • 启用虚拟机的客户机自定义；

&emsp;&emsp;其中我认为必不可少的功能是主机和虚拟机之间复制粘贴，自由的鼠标移动，Unity功能，文件或磁盘共享。强烈建议给虚拟机子系统安装都安装上VMware Tools。需要 注意 的是，VMware是针对虚拟机子系统的，可以他它理解为一种驱动，因此每个虚拟机子系统都需要单独安装一遍VMware Tools。

2 安装VMware

&emsp;&emsp;先启动要安装VMware Tools的虚拟机系统，在VMware客户端菜单栏中找到 虚拟机(M) 菜单栏下的 安装 VMware Tools(T) 并点击。

&emsp;&emsp;然后打开 我的电脑 ，选择 DVD 驱动器（E:）VMware Tools 双击开始安装驱动。

&emsp;&emsp;装驱动一路点下一步就可以了。
