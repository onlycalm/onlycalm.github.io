Vim基础篇（十五）——环绕字符插件vim-surround

前言：
&emsp;&emsp;代码编辑中有很多时候要编辑环绕字符，比如””，’’，()，[]，{}等，通过vim-surround插件可以便捷的对这些符号进行操作。

1 安装vim-surround

&emsp;&emsp;插件 vim-surround 来提供功能齐全的环绕字符操作，有了它将大大提高代码编辑的效率。

3 用法讲解

&emsp;&emsp;需要注意的是，非对称的环绕符比如 ()，[]，{}，使用左符号是会自动在字符串前后插入空格，使用右符号则不插入空格。这种情况也适用于vim-surround其他模式下。
&emsp;&emsp;下面的指令还可与vim的其他跳转指令搭配使用，比如。

指令 含义
e 往后跳转到字符串末
b 往前跳转到字符串首
w 往后跳转到字符串首
$ 往后跳转到最后一列
^ 往前跳转到行首
0 往前跳转到第一列

&emsp;&emsp;与vim-surround指令的搭配比如： yse，ys$ 等等，

3.1 普通模式下

3.1.1 添加环绕符

3.1.1.1 字符串添加环绕符

&emsp;&emsp;通过指令 ysiw<char> 可实现将光标下字符串两侧添加环绕符。

指令 字符串修改前 字符串修改后
ysiw” Calm “Calm”
ysiw( Calm ( Calm )
ysiw) Calm (Calm)

3.1.1.2 行添加环绕符

&emsp;&emsp;通过指令 yss<char> 可实现将光标所在行两侧添加环绕符。

指令 行文本修改前 行文本修改后
yss” Calm Calm Calm “Calm Calm Calm”
yss( Calm Calm Calm ( Calm Calm Calm )
yss) Calm Calm Calm (Calm Calm Calm)

3.1.2 删除环绕符

&emsp;&emsp;通过指令 ds<char> 可实现删除光标两侧最近的环绕符。

指令 字符串修改前 字符串修改后
ds” “Calm” Calm

3.1.3 修改环绕符

&emsp;&emsp;通过指令 cs<old char><new char> 可实现修改光标两侧最近的环绕符。

指令 字符串修改前 字符串修改后
cs”‘ “Calm” ‘Cal ‘

3.2 可视模式下

&emsp;&emsp;可视模式下输入 S<char> 即可实现在所选文本两侧添加环绕字符串。
