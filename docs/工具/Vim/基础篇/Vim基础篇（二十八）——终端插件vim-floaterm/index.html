Vim基础篇（二十八）——终端插件vim-floaterm

前言：
&emsp;&emsp; 在linux下编程，终端最是常用的工具，将Vim和Terminal配合好使用将大大提高效率。

1 Vim使用终端的方式

1.1 暂停Vim

&emsp;&emsp;在Vim界面，按下 ctrl z 会暂停当前的vim，回到Terminal界面，可以在terminal中执行命令，需要回到Vim则在Terminal中输入 fg 即可，Vim会保持退出前的状态。

1.2 内嵌终端

&emsp;&emsp;Vim已集成了Terminal，在Vim中执行指令 :term 会自动分割一个Terminal窗口，在其中执行指令，需要退出终端窗口，在终端中执行 exit 。以分割窗口执行终端的好处是，对窗口的操作都适用。
