4 生成compile_commands.json

&emsp;&emsp;ale默认会解析工程build目录下的compile_commands.json文件，该文件在linux下可由cmake编译工程自动生成，但使用cmake编译需要安装cmake工具并且为工程编写CMakeLists.txt文件。在CMakeLists.txt文件中会给出引用的头文件和源文件路径，因此语法查错时就能根据生成的compile_commands.json文件找到引用的头文件和源文件。compile_commands.json文件默认放到工程根目录或 ./build/ 目录下有效。
&emsp;&emsp;这里不讲CMake的语法规则，可以阅读我的CMake教程或 [CMake](https://cmake.org “CMake””) 官网学习。
