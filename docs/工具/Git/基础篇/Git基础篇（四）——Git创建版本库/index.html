GitеҹәзЎҖзҜҮпјҲеӣӣпјүвҖ”вҖ”GitеҲӣе»әзүҲжң¬еә“

еүҚиЁҖпјҡ
&emsp;&emsp;Gitе№¶дёҚжҳҜеҜ№д»»ж„Ҹзӣ®еҪ•йғҪиғҪиҝӣиЎҢз®ЎзҗҶпјҢйҰ–е…Ҳз®ЎзҗҶзҡ„зӣ®еҪ•еҝ…йЎ»жҳҜGitеҲӣе»әзҡ„зүҲжң¬еә“пјҢйӮЈд№ҲеҰӮдҪ•е°ҶйЎ№зӣ®зӣ®еҪ•еҲӣе»әдёәGitзүҲжң¬еә“иҝӣиЎҢз®ЎзҗҶе‘ўпјҹ

1 еҲӣе»әGitзүҲжң¬еә“

&emsp;&emsp;еңЁе·ҘзЁӢзҡ„ж №зӣ®еҪ•пјҲеҸҜд»ҘжҳҜз©әзӣ®еҪ•пјүдёӢжү§иЎҢвҖңGit BashвҖқе‘Ҫд»ӨпјҢжҲ–еҜ№е·ҘзЁӢж №зӣ®еҪ•ж–Үд»¶еӨ№еҸій”®йј ж ҮиҸңеҚ•дёӯзӮ№еҮ»вҖңGit Bash HereвҖқеј№еҮәе‘Ҫд»ӨиЎҢзӘ—еҸЈжү§иЎҢд»ҘдёӢе‘Ҫд»Өпјҡ

1.1 git init

&emsp;&emsp;иҜҘе‘Ҫд»ӨдјҡеңЁж №зӣ®еҪ•дёӢиҮӘеҠЁеҲӣе»әж–Үд»¶еӨ№вҖқ.gitвҖңпјҲиҜҘзӣ®еҪ•й»ҳи®ӨжҳҜйҡҗи—Ҹзҡ„пјүпјҢж„Ҹе‘ізқҖGitжҠҠиҝҷдёӘзӣ®еҪ•еҸҳдёәдәҶGitеҸҜд»Ҙз®ЎзҗҶзҡ„д»“еә“дәҶпјҢвҖқ.gitвҖңзӣ®еҪ•дёӯи®°еҪ•дәҶGitеҜ№дәҺиҜҘйЎ№зӣ®зҡ„жүҖжңүзүҲжң¬з®ЎзҗҶи®°еҪ•пјҢжүҖд»ҘеҚғдёҮдёҚиҰҒйҡҸж„ҸеҺ»дҝ®ж”№е®ғпјҢдҪ еҸӘйңҖиҝҗиЎҢжҢҮд»Өи®©GitиҮӘеҠЁз®ЎзҗҶиҝҷдёӘзӣ®еҪ•гҖӮ
&emsp;&emsp;еҸҰеӨ–пјҢеҖјеҫ—дёҖжҸҗзҡ„жҳҜпјҢGitж“…й•ҝзҡ„жҳҜж–Үеӯ—ж–Үжң¬зҡ„зүҲжң¬з®ЎзҗҶпјҢеҰӮжһңж¶үеҸҠеӣҫзүҮгҖҒи§Ҷйў‘гҖҒpdfгҖҒзЎ¬д»¶еҺҹзҗҶеӣҫзӯүжӣҙеӨҚжқӮзҡ„ж–Үд»¶пјҢGitеҜ№йҪҗзҡ„зүҲжң¬з®ЎзҗҶж•ҲжһңеҫҲе·®гҖӮ
