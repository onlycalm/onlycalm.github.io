Ubuntu安装wine运行Windows应用

前言：
&emsp;&emsp;在Ubuntu下运行Windows应用，可以通过安装wine实现。

1 wine介绍

&emsp;&emsp;wine让你可以在Ubuntu上运行Windows应用，大多数应用都能很好的支持并运行，比如微信。
&emsp;&emsp;安装方法很简单，可以直接通过系统自带的软件中心安装或终端命令安装。

2 安装wine

2.1 软件中心安装

&emsp;&emsp;打开Ubuntu自带的软件中心，搜索 wine 。

&emsp;&emsp;在搜索结果中选中 wine ，点击 Install ，等待安装完成（国内安装可能较慢，需要耐性等待）。

&emsp;&emsp;通过软件中心安装的wine是版本比较老的稳定版，如果要使用较新的版本，可通过命令行安装。

2.2 指令安装

&emsp;&emsp;添加存储库，在Ubuntu20.04和22.04上测试过。

&emsp;&emsp;输入以下命令安装Wine。

&emsp;&emsp;可通过指令查看wine版本号。

&emsp;&emsp;最后执行一次 winecfg 指令，将使wine创建并部署Windows环境在 home/calm/.wine 路径下，不做任何修改关闭界面。

3 安装Windows应用

&emsp;&emsp;安装的指令为 wine <exe> ，比如安装微信，下载好exe文件后在当前路径下终端执行指令 wine WeChatSetup.exe 。
&emsp;&emsp;勾选接受用户协议，路径不用设置，点击安装即可，通过扫码登录。

&emsp;&emsp;需要注意一点，用wine安装应用不要加sudo，如果用 sudo wine <exe> ，wine会在 root/.wine 又部署一套wine环境并将应用安装到这个路径里，安装在这里后不会创建启动图标，给启动wine安装的应用造成麻烦，所以间隔不带sudo的安装，即安装到 home//.wine 路径下。

4 启动应用

&emsp;&emsp;通过指令 wine + 应用名.exe，可以启动通过wine安装的Windows应用。比如 wine wechat.exe 。
&emsp;&emsp;如果启动应用报关于winbind的错误，比如 0a24:err:winediag:SECUR32_initNTLMSP ntlm_auth was not found or is outdated. Make sure that ntlm_auth >= 3.0.25 is in your path. ，执行以下指令。

5 卸载应用

&emsp;&emsp;要卸载wine安装的应用，命令行输入 sudo wine uninstaller 弹出应用卸载界面进行卸载。

&emsp;&emsp;选择要卸载的应用，点击 Modify/Remove 进行卸载。

6 中文乱码

&emsp;&emsp;安装一些带中文的应用后，中文可能出现显示乱码，中文乱码主要是系统的Language不是中文导致，因此可将系统Language设置为中文，或者wine启动时指定语言格式，比如。

&emsp;&emsp;最好的办法是从真正的Windows系统上将目录 c:\windows\Fonts 内的字体文件全部拷贝到wine的相同路径下 home//.wine/drive_c/windows/Fonts ，注销并重新登录账户即可。
