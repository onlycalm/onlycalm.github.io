自娱自乐，我的博客（九）——Hexo博客添加看板卡通人物

前言：
&emsp;&emsp;为了增强博客的互动性，可以为你的Hexo添加有趣的卡通人物。这些卡通人物可以与鼠标互动，显得更有趣味性。Live2d这款插件使得添加卡通人物非常容易，并且提供了很多制作精良的人物模型供选择。

1 安装插件和卡通模型

1.1 安装插件

&emsp;&emsp;在博客根目录下执行cmd指令安装插件EYHN/hexo-helper-live2d。

1.2 安装模型

&emsp;&emsp;hexo-helper-live2d插件为我们提供了很多精美模型选择，可以访问插件作者博客预览模型，选择你喜欢的模型后在博客根目录下执行cmd指令安装这个模型，比如我安装名为 unitychan 的模型。更多的模型名称请查阅模型作者的Github

&emsp;&emsp;将模型名替换为你自己喜欢的。
