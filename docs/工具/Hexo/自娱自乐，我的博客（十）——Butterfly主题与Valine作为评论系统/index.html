自娱自乐，我的博客（十）——Butterfly主题与Valine作为评论系统

前言：
&emsp;&emsp;评论留言功能时博客互动的主要方式，可以说这是一个合格博客的必备功能。如何为自己博客搭建一个免费的评论系统呢？Valine就是这样一种免费且简单易用的评论系统。它不要求评论者登录，只需要填写用户名和邮箱即可。在之前的文章中我介绍过Gitalk作为评论系统，遗憾的是国内访问速度太慢了影响到正常使用。

1 LeanCloud

1.1 注册LeanCloud

&emsp;&emsp;国内 LeanCloud 现在用于评论系统也需要域名备案了，所以如果你的域名没有备案可以使用 LeanCloud国际版 。
&emsp;&emsp;登录LeanCloud注册账号。注意国内版和国际版账号目前是不通用的，需要分别注册。需要绑定邮箱和手机，为了能创建应用必须得实名认证，好在支持支付宝扫码快捷认证。

1.2 创建应用

&emsp;&emsp;注册账号并实名认证后需要创建应用并配置。

&emsp;&emsp;选择免费的 开发版。

&emsp;&emsp;应用创建好后在设置中可以看到该应用的 AppID 和 AppKey，在后面Butterfly主题中将会用到它们。

1.3 绑定域名

&emsp;&emsp;我们为刚才创建的应用绑定域名。再次提醒，国内LeanCloud只能绑定备案过的域名，而国际版绑定的域名不需要备案。

&emsp;&emsp;还需要到你的域名服务商处的控制台中添加一条 CNAME 记录。添加完成后只需要等待LeanCloud审核通过即可，可能需要半小时到一小时左右，请耐心等待。也可能申请失败，请多次尝试（我试了三次才成功）。

2 配置Butterfly，开启Valine评论功能

&emsp;&emsp;Butterfly主题从v3.0.0开始支持双评论系统了，可以开启使用两种方式评论，在配置文件butterfly.yml中修改配置。

&emsp;&emsp;对Valine进行配置，同样在配置文件butterfly.yml中修改配置。

&emsp;&emsp;appID 和 appKey 就是之前创建应用时获得的，将其填入配置中。avatar 可以设置评论者默认使用的头像，可以在 Valine 官网上查看可以设置哪些头像。
&emsp;&emsp;好了，一切就绪可以去重新部署试试效果了。

3 管理评论

&emsp;&emsp;登录LeanCloud后找到创建的管理评论的应用即可对评论进行管理。
