Leaderf的安装与配置使用

前言：
&emsp;&emsp;Leaderf使得Vim可以实现像IDE那样的模糊搜索功能，这篇文章介绍配置和使用的方法。

1 下载安装Leaderf

&emsp;&emsp;如果你安装了Vundle, 那么直接再Vimrc中添加指令Plugin 'Yggdroot/LeaderF'等待安装完成即可，如果你没有装Vundle，那直接去github上下载并解压拷贝即可。

&emsp;&emsp;需要注意的是Leaderf还依赖与ctags和rg（这个插件全名是Rigrep）插件，如果你还没有装可以去github上直接下载Release版的，将其中的*.exe文件拷贝到与GVim.exe同级目录即可。

2 Leaderf常用指令

2.1 查询文件：:Leaderf file

&emsp;&emsp;默认是从根目录内的文件中查找。

2.2 查询函数: :Leaderf function

&emsp;&emsp;默认是再当前文件中查找函数。

2.3 模糊查询字符串: Leaderf rg

&emsp;&emsp;默认从根目录内的文件中查找，模糊查找，非常只能且迅速。

2.4 查询最近打开过的文件: :Leaderf mru

&emsp;&emsp;这个功能特别是在你关闭vim后，下次再打开继续编辑时很有用。

2.5 查询Buffer: :Leaderf buffer
