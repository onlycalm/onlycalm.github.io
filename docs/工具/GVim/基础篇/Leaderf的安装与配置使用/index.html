Leaderfзҡ„е®үиЈ…дёҺй…ҚзҪ®дҪҝз”Ё

еүҚиЁҖпјҡ
&emsp;&emsp;LeaderfдҪҝеҫ—VimеҸҜд»Ҙе®һзҺ°еғҸIDEйӮЈж ·зҡ„жЁЎзіҠжҗңзҙўеҠҹиғҪпјҢиҝҷзҜҮж–Үз« д»Ӣз»Қй…ҚзҪ®е’ҢдҪҝз”Ёзҡ„ж–№жі•гҖӮ

1 дёӢиҪҪе®үиЈ…Leaderf

&emsp;&emsp;еҰӮжһңдҪ е®үиЈ…дәҶVundle, йӮЈд№ҲзӣҙжҺҘеҶҚVimrcдёӯж·»еҠ жҢҮд»ӨPlugin 'Yggdroot/LeaderF'зӯүеҫ…е®үиЈ…е®ҢжҲҗеҚіеҸҜпјҢеҰӮжһңдҪ жІЎжңүиЈ…VundleпјҢйӮЈзӣҙжҺҘеҺ»githubдёҠдёӢиҪҪе№¶и§ЈеҺӢжӢ·иҙқеҚіеҸҜгҖӮ

&emsp;&emsp;йңҖиҰҒжіЁж„Ҹзҡ„жҳҜLeaderfиҝҳдҫқиө–дёҺctagsе’ҢrgпјҲиҝҷдёӘжҸ’д»¶е…ЁеҗҚжҳҜRigrepпјүжҸ’д»¶пјҢеҰӮжһңдҪ иҝҳжІЎжңүиЈ…еҸҜд»ҘеҺ»githubдёҠзӣҙжҺҘдёӢиҪҪReleaseзүҲзҡ„пјҢе°Ҷе…¶дёӯзҡ„*.exeж–Үд»¶жӢ·иҙқеҲ°дёҺGVim.exeеҗҢзә§зӣ®еҪ•еҚіеҸҜгҖӮ

2 Leaderfеёёз”ЁжҢҮд»Ө

2.1 жҹҘиҜўж–Үд»¶пјҡ:Leaderf file

&emsp;&emsp;й»ҳи®ӨжҳҜд»Һж №зӣ®еҪ•еҶ…зҡ„ж–Үд»¶дёӯжҹҘжүҫгҖӮ

2.2 жҹҘиҜўеҮҪж•°: :Leaderf function

&emsp;&emsp;й»ҳи®ӨжҳҜеҶҚеҪ“еүҚж–Үд»¶дёӯжҹҘжүҫеҮҪж•°гҖӮ

2.3 жЁЎзіҠжҹҘиҜўеӯ—з¬ҰдёІ: Leaderf rg

&emsp;&emsp;й»ҳи®Өд»Һж №зӣ®еҪ•еҶ…зҡ„ж–Үд»¶дёӯжҹҘжүҫпјҢжЁЎзіҠжҹҘжүҫпјҢйқһеёёеҸӘиғҪдё”иҝ…йҖҹгҖӮ

2.4 жҹҘиҜўжңҖиҝ‘жү“ејҖиҝҮзҡ„ж–Үд»¶: :Leaderf mru

&emsp;&emsp;иҝҷдёӘеҠҹиғҪзү№еҲ«жҳҜеңЁдҪ е…ій—ӯvimеҗҺпјҢдёӢж¬ЎеҶҚжү“ејҖз»§з»ӯзј–иҫ‘ж—¶еҫҲжңүз”ЁгҖӮ

2.5 жҹҘиҜўBuffer: :Leaderf buffer

&emsp;&emsp;еҪ“еүҚbufferдёҖи§Ҳзңјеә•пјҢеҫҲзҲҪгҖӮ

2.6 жҢүй”®жҳ е°„

&emsp;&emsp;еҪ“з„¶пјҢжңҖеҘҪиҝҳжҳҜжҢүй”®жҳ е°„дёҖдёӢпјҢжҜ•з«ҹиҝҷеҮ дёӘжҗңзҙўиҝҳжҳҜз»Ҹеёёз”ЁеҲ°зҡ„пјҢжҜҸж¬Ўжү“иҝҷд№ҲеӨҡеӨӘзҙҜгҖӮжҲ‘зҡ„й…ҚзҪ®еҰӮдёӢпјҡ

&emsp;&emsp;PS: е…¶е®һдҪ иҰҒжҳҜжҮ’еҫ—и®°жҢҮд»ӨпјҢдҪ еҸҜд»ҘдёҚз®ЎжҹҘж–Үд»¶иҝҳжҳҜеҮҪж•°еӯ—з¬ҰзӯүйғҪз”ЁжҢҮд»ӨLeaderf rgе°ұиЎҢпјҢеҪ“з„¶и®ҫзҪ®дёәеҝ«жҚ·й”®дјҡжҜ”иҫғеҘҪгҖӮ
