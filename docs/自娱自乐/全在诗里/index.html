自娱自乐，做大娱乐家

前言：
  生活不易，多才多艺。

——生生灯火，明暗无辄

2020-05-05

《情满意足》
——大娱乐家
人生迟不复，逆行金银路。
多情寻旧故，岁月太残酷。
欢宴座渐疏，笑别心涩苦。
祝君锦似虎，勿忘情灌注。

2020-04-11

《迷途》
——大娱乐家
少年立志与命夺，待成青帝欲报国。
年过二三外漂泊，如遇阻绊必奋博。
奈何愚钝智笨拙，松甲泄气梦浑浊。
今虽志短为生活，老兵重振器打磨。

2019-02-12

《小城故事》
——大娱乐家
疫情无情人有情，小城不拒自家人。
面罩难挡人面善，生活静好何需惧。
俯身即拾攀枝花，家中又添旧菜肴。
蜿蜒长路绕青山，斜阳无遮映江红。
远眺恰似盆中城，近观却成井底村。
小城日新未献力，索取安宁又作别。

2019-01-28

《留感》
——大娱乐家
疑因流感或冠毒，寒夜咳声不绝屋。
安宁入眠噩梦阻，现实梦境均是恶。
起身找寻快乐处，愉悦心情必康复。
满屏新闻满屏负，作罢蹒跚回夜宿。

2020-01-20

《叹谈》
——大娱乐家
晨露未聚点灯添衣，霞光稍显轻装出行。
扶摇直上彩云之南，归家路上无谓功名。
生百家姓得重复命，翻万座山回牵挂地。
蜉蝣之力难撼天地，鲲鹏之志气吞万里。

2019-10-06

《回家》
——大娱乐家
喜逢国事乐，久未归家情。
行囊空空也，千里一日达。
山水犹在卧，不复当年景。
亲朋安居业，不同往日习。
父白双鬓发，母浊双目朦。
久未成大事，幸牵姻缘人。
匆匆五日娱，忙忙行程路。
叹息人生命，再别小镇情。

2019-09-06

《再到佳节，何以解忧？》
——大娱乐家
金比情坚，命比草贱。
梦如流沙，运如浮萍。
回不去的家乡，盼不完的亲人。
数不尽的心酸，到不了的理想。
人生终为空，甘做尘土即错？
财富难入土，奋求荣华乃对？
对错难辩，似是而非。
三丈房，三尺台，三寸黑白归尘土。

2019-06-29

无题
——大娱乐家

  公司聚餐回来太渴，因渴故烧水，因倦而睡去。夜闻怪味猛惊醒，热水器通红蹦火苗，三步并作两步走，力拔山河誓灭插座。室内篝火似野营，熏烟弥漫盘旋如黑龙。速舀凉水泼火，速开空调去味，速开电灯去光。恐对舍见火，惧邻舍察味，阳台里来回踱步，内心里七上八下。细观壁白如纸，精审地镜如玉，待真无患而安。事发凌晨一点半，谢佛留薄命，吓死爷爷了！！！已困，回笼觉续。
