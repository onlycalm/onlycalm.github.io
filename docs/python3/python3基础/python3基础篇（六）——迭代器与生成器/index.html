еүҚиЁҖпјҡ
йҳ…иҜ»иҝҷзҜҮж–Үз« жҲ‘иғҪеӯҰеҲ°д»Җд№Ҳпјҹ
  иҝҷзҜҮж–Үз« е°ҶдёәдҪ д»Ӣз»Қpython3зҡ„иҝӯд»ЈеҷЁе’Ңз”ҹжҲҗеҷЁд»ҘеҸҠеҲ—иЎЁз”ҹжҲҗејҸзҡ„иҜҰз»Ҷз”Ёжі•гҖӮ

1 иҝӯд»ЈеҷЁ

  еңЁpython3дёӯеҸҜд»ҘдёәеәҸеҲ—еҲӣе»әиҝӯд»ЈеҷЁгҖӮиҝӯд»ЈеҷЁжҳҜдёҖдёӘз”ЁдәҺи®°еҪ•еңЁеәҸеҲ—дёӯеҪ“еүҚйҒҚеҺҶдҪҚзҪ®зҡ„з»“жһ„пјҢжҢҒз»ӯи®ҝй—®е®ғе°ҶиғҪдҫқж¬Ўи®ҝй—®еәҸеҲ—зҡ„жүҖжңүе…ғзҙ гҖӮе®ғд»ҺеәҸеҲ—зҡ„з¬¬дёҖдёӘе…ғзҙ ејҖе§Ӣи®ҝй—®пјҢзӣҙеҲ°и®ҝй—®е®ҢжүҖжңүе…ғзҙ гҖӮиҝӯд»ЈеҷЁеҸӘиғҪеүҚиҝӣдёҚиғҪеҗҺйҖҖпјҢд№ҹеҚіи®ҝй—®иҝҮзҡ„е…ғзҙ дёҚиғҪеҶҚж¬Ўи®ҝй—®гҖӮеёёз”Ёзҡ„ж–№жі•жңүiter()з”ЁдәҺеҲӣе»әиҝӯд»ЈеҷЁпјҢnext()еҮҪж•°иҝ”еӣһеҪ“еүҚе…ғзҙ е№¶жҢҮеҗ‘дёӢдёҖдёӘе…ғзҙ гҖӮ
д»Јз ҒзӨәдҫӢпјҡ

иҝҗиЎҢз»“жһңпјҡ

  еҸҜд»Ҙдёәеӯ—з¬ҰдёІгҖҒе…ғз»„гҖҒеҲ—иЎЁгҖҒйӣҶеҗҲгҖҒеӯ—е…ёеҲӣе»әиҝӯд»ЈеҷЁпјҢд№ҹеҸҜд»ҘжҠҠдёҖдёӘзұ»дҪңдёәиҝӯд»ЈеҷЁпјҢжҲ–иҖ…иҜҙе®ғд»¬жҳҜвҖңеҸҜд»Ҙиҝӯд»Јзҡ„вҖқпјҢдёӢйқўиҰҒи®Ізҡ„з”ҹжҲҗеҷЁд№ҹжҳҜеҸҜд»Ҙиҝӯд»Јзҡ„гҖӮиҝӯд»ЈеҷЁд№ҹеҸҜе’ҢforеҫӘзҺҜй…ҚеҗҲдҪҝз”ЁгҖӮ
д»Јз ҒзӨәдҫӢпјҡ

иҝҗиЎҢз»“жһңпјҡ

2 з”ҹжҲҗејҸ

  и®Із”ҹжҲҗеҷЁд№ӢеүҚиҝҳйңҖиҰҒиЎҘе……дёҖдёӢpython3дёӯдёҖдёӘеҫҲејәзҡ„зү№жҖ§еҚівҖңз”ҹжҲҗејҸвҖқгҖӮеңЁеҲӣе»әеҲ—иЎЁж—¶пјҢе…Ғи®ёе°Ҷи§„еҲҷеҶҷеңЁеҲ—иЎЁдёӯпјҢpython3е°ҶдјҡжҢүз…§жӯӨи§„еҲҷиҮӘеҠЁжҺЁеҜјеҮәеҲ—иЎЁзҡ„е…ғзҙ гҖӮиҝҷж— з–‘жҳҜдёҖз§ҚеҲӣе»әеҲ—иЎЁзҡ„з®ҖжҙҒеҶҷжі•гҖӮ

2.1 дёәд»Җд№ҲйңҖиҰҒвҖңз”ҹжҲҗејҸвҖқпјҹ

  еҒҮи®ҫи®©дҪ еҲӣе»әиҝҷж ·дёҖдёӘеҲ—иЎЁпјҢд»–зҡ„е…ғзҙ з¬ҰеҗҲе…¬ејҸ$y=x^2+x$пјҢxдёәиҮӘз„¶ж•°гҖӮеҰӮжһңе‘ҠиҜүдҪ еҸӘйңҖиҰҒеүҚдә”йЎ№пјҢдҪ еҸҜд»ҘеҫҲиҪ»жқҫзҡ„еҲӣе»әеҮәиҝҷдёӘеҲ—иЎЁ

  еҰӮжһңжҲ‘иҰҒжұӮеҶҷеҮәеүҚ10йЎ№е‘ўпјҹеҪ“з„¶д№ҹиғҪеҶҷеҮәпјҢдёҚиҝҮиҝҷж¬ЎиҰҒеәҹдёҖзӮ№еҠҹеӨ«еҺ»и®Ўз®—гҖӮ

  еҰӮжһңжҳҜеүҚ100йЎ№е‘ўпјҹдҪ еҸҜиғҪдјҡе…ҲеҲӣе»әдёҖдёӘз©әеҲ—иЎЁпјҢз„¶еҗҺеҶҷдёҖдёӘеҫӘзҺҜеҮҪж•°еҺ»и®Ўз®—жІЎдёҖдёӘе…ғзҙ е№¶е°Ҷе…¶иЈ…е…ҘеҲ—иЎЁдёӯгҖӮд»Јз ҒиҝҷдёӘж ·еӯҗпјҡ

  еҰӮжһңеңЁpython3дёӯиҝҷж ·еҶҷе°ҶдјҡйқһеёёвҖқз¬ЁвҖңзҡ„еҶҷжі•гҖӮеҸҜд»ҘеҲ©з”Ёз”ҹжҲҗејҸиҝҷж ·еҲӣе»әеҲ—иЎЁпјҡ

  жҲ‘д»¬е°Ҷе…¬ејҸ$x^2+x$зӣҙжҺҘеҶҷеңЁдәҶеҲ—иЎЁзҡ„[]еҶ…пјҢе№¶дё”дҪҝз”ЁдәҶforе’Ңrange()еҮҪж•°йҷҗе®ҡдәҶxзҡ„еҸ–еҖјиҢғеӣҙгҖӮиҝҷж ·python3е°ұеҸҜд»Ҙж №жҚ®иҝҷдәӣи§„еҲҷеҺ»иҮӘеҠЁжҺЁеҜјеҮәеҲ—иЎЁдёӯзҡ„жҜҸдёҖдёӘе…ғзҙ гҖӮдҪ еҸҜд»Ҙе°Ҷ[]еҶ…зҡ„д»Јз ҒеҲҶжҲҗдёӨйғЁеҲҶжқҘзңӢпјҢдёҖйғЁеҲҶжҳҜз”ҹжҲҗи§„еҲҷx * x + xпјҢеҸҰдёҖйғЁеҲҶжҳҜеҫӘзҺҜз»“жһ„зҡ„иҢғеӣҙйҷҗе®ҡfor x in range(100)гҖӮ
  еҲ—иЎЁз”ҹжҲҗејҸдё»иҰҒдҪңз”Ёе°ұжҳҜдҪҝеҫ—еҲӣе»әеҲ—иЎЁеҸҳеҫ—жӣҙз®ҖдҫҝгҖӮ
  еҶҚжҖқиҖғдёҖдёӘй—®йўҳпјҢеҰӮжһңиҰҒжұӮеҲӣе»әзҡ„еҲ—иЎЁе…ғзҙ еҢ…еҗ«еүҚдёҖеҚғпјҢз”ҡиҮідёҖдёҮйЎ№е‘ўпјҹеҪ“е…ғзҙ дёӘж•°еәһеӨ§зҡ„ж—¶еҖҷпјҢдјҡеёҰжқҘеҸҰеӨ–дёҖдёӘй—®йўҳпјҢе°ұжҳҜеҶ…еӯҳејҖй”ҖиҝҮеӨ§гҖӮиҝҷж—¶еҖҷжҲ‘д»¬еҸҜд»ҘзүәзүІдёҖзӮ№ж—¶й—ҙж•ҲзҺҮеҺ»иҠӮзңҒз©әй—ҙж•ҲзҺҮгҖӮи®©еҲ—иЎЁеҸӘи®°еҪ•и®Ўз®—и§„еҲҷпјҲе…¬ејҸпјүпјҢеҪ“йңҖиҰҒи®ҝй—®е…ғзҙ ж—¶е®һж—¶и®Ўз®—з”ҹжҲҗпјҢдёҚйңҖиҰҒдҝқеӯҳжүҖжңүе…ғзҙ пјҢиҝҷе°ұжҳҜеҗҺйқўиҰҒи®ІеҲ°зҡ„з”ҹжҲҗеҷЁзҡ„дҪңз”ЁгҖӮ

2.2 еҲ—иЎЁз”ҹжҲҗејҸиҜӯжі•и§„еҲҷ

  жҙ»з”Ёз”ҹжҲҗејҸе°ҶдјҡдҪҝеҫ—д»Јз Ғжӣҙз®ҖжҙҒжңүж•ҲгҖӮ

2.2.1 з”ҹжҲҗејҸз»ҷеҮәи§„еҲҷе’ҢиҢғеӣҙ

  дёҠйқўжҲ‘д»¬е·Із»ҸзңӢеҲ°дәҶеҲ—иЎЁз”ҹжҲҗејҸдёӯз»ҷеҮәз”ҹжҲҗи§„еҲҷпјҲе…¬ејҸпјүе’ҢеҸӮж•°иҢғеӣҙзҡ„иҜӯжі•з»“жһ„дәҶгҖӮеҶҚеӣһйЎҫдёҖдёӢпјҡList = [x * x + x for x in range(100)]
  жҺҘдёӢжқҘд»Ӣз»ҚжӣҙеӨҚжқӮдёҖзӮ№зҡ„з”Ёжі•д»Ҙж»Ўи¶іжӣҙеӨҡзҡ„и®ҫи®ЎйңҖжұӮгҖӮ

2.2.2 forиҜӯеҸҘд№ӢеҗҺеҠ е…ҘifиҜӯеҸҘ

  еҜ№дәҺдёҠйқўзҡ„еҲ—иЎЁжҲ‘д»¬ж·»еҠ дёҖдёӘи§„еҲҷпјҢе°ұжҳҜе…ғзҙ ж—ўиҰҒз¬ҰеҗҲе…¬ејҸ$y=x^2+x$пјҢxдёәиҮӘз„¶ж•°пјҢиҖҢдё”иҰҒжұӮxеҝ…йЎ»иў«иғҪ5ж•ҙйҷӨгҖӮдёәдәҶж»Ўи¶іиҝҷдёӘиҰҒжұӮжҲ‘д»¬йңҖиҰҒеҜ№е…ғзҙ иҝӣиЎҢ зӯӣйҖүгҖӮдҪҝз”ЁifиҜӯеҸҘиҝӣиЎҢзӯӣйҖүгҖӮ
д»Јз ҒзӨәдҫӢпјҡ

иҝҗиЎҢз»“жһңпјҡ

  зӯӣйҖүеҗҺ100дёӘе…ғзҙ еҸӘеү©дёӢ20дёӘдәҶгҖӮеҸӘжҳҜеңЁ[]йҮҢforиҜӯеҸҘд№ӢеҗҺж·»еҠ дәҶдёҖеҸҘif x % == 0е°ұеҜ№е…ғзҙ иҝӣиЎҢдәҶзӯӣйҖүпјҢеҸӘз•ҷдёӢз¬ҰеҗҲжқЎд»¶зҡ„е…ғзҙ гҖӮ
  д№ҹеҸҜд»ҘеңЁforиҜӯеҸҘд№ӢеҗҺдҪҝз”ЁеӨҡдёӘifпјҢи®ҫзҪ®еӨҡдёӘзӯӣйҖүжқЎд»¶гҖӮ
  дёӢйқўзҡ„д»Јз ҒзӯӣйҖүжқЎд»¶дёәиғҪиў«5е’Ң2ж•ҙйҷӨзҡ„е…ғзҙ гҖӮ
д»Јз ҒзӨәдҫӢпјҡ

иҝҗиЎҢз»“жһңпјҡ

  з”ҹжҲҗејҸдёӯforиҜӯеҸҘд№ӢеҗҺеҠ е…ҘifиҜӯеҸҘдәӢе…ҲеҜ№е…ғзҙ зҡ„зӯӣйҖүпјҢйңҖиҰҒ ==жіЁж„Ҹ== зҡ„жҳҜдёҚиғҪдҪҝз”Ёif-elseгҖҒif-elifгҖҒif-elif-elseиҝҷдәӣз»“жһ„пјҢеҸӘиғҪдҪҝз”Ёifз»“жһ„гҖӮ

2.2.3 forиҜӯеҸҘд№ӢеүҚеҠ е…ҘifиҜӯеҸҘ

  forиҜӯеҸҘеүҚйқўдҪҝз”Ёif-elseиҜӯеҸҘпјҢе®ғе…¶е®һжҳҜз”ҹжҲҗи§„еҲҷпјҲе…¬ејҸпјүзҡ„дёҖйғЁеҲҶпјҢдҪ еҸҜд»Ҙз®ҖеҚ•зҗҶи§ЈдёәеҲҶж®өеҮҪж•°гҖӮиҰҒжіЁж„ҸеҸӘиғҪдҪҝз”Ёif-elseз»“жһ„гҖӮ
  жҲ‘д»¬еҜ№дёҠйқўзҡ„дҫӢеӯҗжҲ‘д»¬еҶҚеўһеҠ дёҖдёӘжқЎд»¶пјҢеүҚ100дёӘе…ғзҙ дёӯпјҢеүҚ50дёӘе…ғзҙ йңҖиҰҒж»Ўи¶іе…¬ејҸy=x^2+xпјҢеҗҺ50дёӘйңҖиҰҒж»Ўи¶іе…¬ејҸy=-x^2+xгҖӮиҝҷжҳҜдёҖдёӘеҲҶж®өеҮҪж•°пјҢж №жҚ®xзҡ„еҖјжҳҜеҗҰеӨ§дәҺ50еҺ»еҶіе®ҡдҪҝз”Ёе“ӘдёӘе…¬ејҸпјҢд№ҹеҚіжңүдёҖдёӘйҖүжӢ©зҡ„еҠЁдҪңпјҢиҝҷе°ұйңҖиҰҒif-elseеҺ»е®һзҺ°гҖӮ
д»Јз ҒзӨәдҫӢпјҡ

иҝҗиЎҢз»“жһңпјҡ

  дёҠйқўзҡ„д»Јз ҒеҪ“ж»Ўи¶іif x % 2 == 0

2.2.4 еӨҡдёӘеҸӮж•°

  дёҠйқўжҲ‘д»¬д»Ӣз»Қзҡ„з”ҹжҲҗејҸйғҪжҳҜдёҖдёӘеҸӮж•°еҚі$y=f(x)$зҡ„еҪўејҸпјҢе®һйҷ…дёҠжҳҜеҸҜд»ҘеӨҡдёӘеҸӮж•°зҡ„пјҢиҝҷйҮҢжҲ‘д»¬д»ҘдёӨдёӘеҸӮж•°дёҫдҫӢгҖӮдёӨдёӘеҸӮж•°зҡ„з”ҹжҲҗејҸйңҖиҰҒж»Ўи¶іе…¬ејҸ$z=f(x,y)$гҖӮжҲ‘д»¬еҒҮи®ҫйңҖиҰҒеҲӣе»әиҝҷж ·дёҖдёӘеәҸеҲ—пјҡxеҸӮж•°жҳҜеҘҮж•°пјҢyеҸӮж•°жҳҜеҒ¶ж•°гҖӮи®Ўз®—5д»ҘеҶ…зҡ„xдёҺyзҡ„е’ҢгҖӮйңҖиҰҒж»Ўи¶іе…¬ејҸ$z=x+y$дё”xвҲҲ[1, 3, 5],yвҲҲ[0, 2, 4]гҖӮ
д»Јз ҒзӨәдҫӢпјҡ

  з®ҖеҚ•и§ЈйҮҠдёӢzip()еҮҪж•°пјҢе®ғжҳҜpythonзҡ„еҶ…зҪ®еҮҪж•°пјҢеңЁиҝҷйҮҢе®ғзҡ„дҪңз”ЁеҗҲе№¶еҲ—иЎЁaе’Ңbз”ҹжҲҗеҲ—иЎЁ[(1, 0), (3, 2), (5, 4)]гҖӮxе’Ңyзҡ„еҖје°ұеҸ–иҮӘиҝҷдёӘж–°е…ғз»„дёӯзҡ„е…ғзҙ пјҢд»ҺиҖҢе®һзҺ°дәҶдёӨдёӘеҸ–еҖјдёҚеҗҢзҡ„еҸӮж•°жұӮе’ҢгҖӮдёүдёӘеҸҠд»ҘдёҠзҡ„еҸӮж•°еҸҜд»Ҙзұ»жҺЁгҖӮ

2.2.4 еӨҡдёӘfor

  еӨҡдёӘforдё»иҰҒз”ЁдәҺе°ҶеӨҡдёӘеҸӮж•°зҡ„еҸ–еҖјиҝӣиЎҢдёҖдёҖз»„еҗҲгҖӮжҜ”еҰӮдёӨдёӘеӯ—з¬ҰдёІ"ABC"е’Ң"abc"е®ғд»¬зҡ„еӯ—з¬Ұд№Ӣй—ҙжңүе“Әдәӣз»„еҗҲпјҹ

  еӯ—з¬ҰдёІзӣёеҠ е°ұжҳҜжҠҠеӯ—з¬ҰдёІжӢјжҺҘгҖӮдёӨдёӘforеҲҶеҲ«йҒҚеҺҶдәҶxе’Ңyзҡ„жүҖжңүеҸҜиғҪеҸ–еҖјпјҢеҲҷжҖ»зҡ„е…ғзҙ еҸҜиғҪе°ұжҳҜдёӨдёӘforйҒҚеҺҶе…ғзҙ зҡ„з»„еҗҲгҖӮдёүдёӘеҸҠд»ҘдёҠдёӘж•°зҡ„forеҸҜд»Ҙзұ»жҺЁгҖӮ
  йҖҡиҝҮдёҠйқўзҡ„еӯҰд№ дҪ еҸҜиғҪд№ҹжғҠеҸ№иҝҷд№ҲеӨҚжқӮзҡ„йҖ»иҫ‘йғҪеҸҜд»ҘеңЁеҲ—иЎЁе®ҡд№үж—¶е°ұжһҒе…¶зІҫз®Җзҡ„е®һзҺ°дәҶпјҢиҝҷзЎ®е®һжҳҜpythonејәеӨ§зҡ„дёҖдёӘең°ж–№гҖӮ

3 з”ҹжҲҗеҷЁ

3.1 дҪҝз”Ё()еҲӣе»әз”ҹжҲҗеҷЁ

  иҝӯд»ЈеҷЁе’ҢеҸҜиҝӯд»Јзҡ„ж•°жҚ®зұ»еһӢе°ҶжүҖжңүе…ғзҙ йғҪж”ҫеңЁеҶ…еӯҳпјҢеҪ“йңҖиҰҒж—¶иҝӣиЎҢи®ҝй—®иҜ»еҸ–пјҢеҰӮжһңе…ғзҙ дёӘж•°еәһеӨ§е°Ҷдјҡйқһеёёж¶ҲиҖ—еҶ…еӯҳгҖӮиҝҷж—¶еҖҷеҸҜд»ҘзүәзүІдёҖзӮ№ж—¶й—ҙиҠӮзңҒз©әй—ҙпјҢжҲ‘д»¬дҪҝз”Ёз”ҹжҲҗеҷЁпјҢе®ғи®°еҪ•зҡ„дёҚжҳҜжҜҸдёӘе…ғзҙ зҡ„еҖјпјҢиҖҢжҳҜз”ҹжҲҗе…ғзҙ зҡ„и§„еҲҷпјҢеҪ“иҝӣиЎҢи®ҝй—®ж—¶йҖҡиҝҮи§„еҲҷе®һж—¶и®Ўз®—еҮәжқҘгҖӮз”ұдәҺз”ҹжҲҗеҷЁд№ҹжҳҜеҸҜиҝӯд»Јзҡ„пјҢжүҖд»Ҙе’Ңиҝӯд»ЈеҷЁдёҖж ·еҸҜд»ҘдҪҝз”Ёnext()гҖҒе’ҢforеҫӘзҺҜиҝӣиЎҢиҝӯд»ЈгҖӮ
  дёӢйқўд»Јз Ғе®һзҺ°е°ҶеҲ—иЎЁе…ғзҙ з”Ёз”ҹжҲҗеҷЁдә§з”ҹпјҢжһ„е»әдәҶдёҖдёӘз”ҹжҲҗеҷЁгҖӮ
д»Јз ҒзӨәдҫӢпјҡ

иҝҗиЎҢз»“жһңпјҡ

  з”ҹжҲҗеҷЁд№ҹжҳҜеҸҜиҝӯд»Јзҡ„пјҢжүҖд»ҘеҸҜдҪҝз”Ёnext()еҮҪж•°иҝӣиЎҢиҝӯд»ЈпјҢд№ҹеҸҜдҪҝз”ЁforеҫӘзҺҜиҝӣиЎҢиҝӯд»ЈгҖӮд№ҹжңүеҫҲеӨҡдәәе°Ҷз”ҹжҲҗеҷЁзңӢеҒҡдёҖз§Қзү№ж®Ҡзҡ„иҝӯд»ЈеҷЁгҖӮ
==з”ҹжҲҗеҷЁе’Ңиҝӯд»ЈеҷЁе…ұеҗҢзӮ№пјҡ==
  йғҪиғҪи®°еҪ•е…ғзҙ зҡ„еҪ“еүҚи®ҝй—®дҪҚзҪ®пјҢйғҪеҸҜиў«з”Ёnext()е’ҢforиҝӣиЎҢиҝӯд»ЈпјҢиҝӯд»ЈеҸӘиғҪеҗ‘еүҚдёҚиғҪйҖҖеҗҺгҖӮ
==з”ҹжҲҗеҷЁе’Ңиҝӯд»ЈеҷЁдёҚеҗҢзӮ№пјҡ==
  еңЁеҲӣе»әдёҠиҝӯд»ЈеҷЁдҪҝз”Ёiter()иҖҢз”ҹжҲҗеҷЁдҪҝз”Ё()жҲ–yieldеҲӣе»әгҖӮеңЁиҝӯд»Јзҡ„иҝҮзЁӢдёӯиҝӯд»ЈеҷЁеҸӘжҳҜжҠҠдәӢе…ҲеӯҳеӮЁзҡ„еҖјиҝ”еӣһпјҢиҖҢз”ҹжҲҗеҷЁйңҖиҰҒжҢүз…§и§„еҲҷеҺ»и®Ўз®—еҗҺиҝ”еӣһгҖӮ

3.2 дҪҝз”ЁyieldеҲӣе»әз”ҹжҲҗеҷЁ

  yieldиў«з”ЁдәҺеҮҪж•°дёӯи°ғз”ЁпјҢдҪҝз”ЁдәҶиҜҘе…ій”®еӯ—зҡ„еҮҪж•°е°ұжҲҗдәҶз”ҹжҲҗеҷЁгҖӮз”ҹжҲҗеҷЁеҮҪж•°дёҺжҷ®йҖҡеҮҪж•°дёҚеҗҢпјҢе®ғиҝ”еӣһзҡ„жҳҜиҝӯд»ЈеҷЁгҖӮyieldжңүзӮ№зұ»дјјreturnпјҢе®ғд»¬иғҪйҖҖеҮәеҮҪж•°е№¶иҝ”еӣһдёҖзӣҙеҖјгҖӮдёҚеҗҢзҡ„жҳҜreturnдёҚдјҡи®°еҪ•еҮҪж•°зҡ„дҝЎжҒҜпјҢиҖҢдҪҝз”Ёyieldдјҡи®°еҪ•дёӢеҮҪж•°еҪ“еүҚиҝҗиЎҢзҡ„жүҖжңүдҝЎжҒҜеҗҺйҖҖеҮәеҮҪж•°е№¶иҝ”еӣһдёҖдёӘеҖјпјҢеҫ…дёӢж¬ЎеҶҚж¬Ўи°ғз”ЁиҝҷдёӘеҮҪж•°ж—¶пјҢз»§з»ӯдҪҝз”Ёдҝқеӯҳзҡ„дҝЎжҒҜе№¶д»ҺдёҠж¬ЎйҖҖеҮәзҡ„yieldдҪҚзҪ®ејҖе§Ӣжү§иЎҢгҖӮйҖҡдҝ—зӮ№иҜҙиҝӯд»ЈеҷЁе°ұжҳҜе®һзҺ°дәҶдёҖдёӘвҖңе»¶иҝҹвҖқеҠҹиғҪпјҢе®ғеҸҜд»Ҙе°ҶеҮҪж•°зҡ„жү§иЎҢжҡӮеҒңгҖӮеҢ…жӢ¬еүҚйқўи®Ізҡ„()еҲӣе»әзҡ„иҝӯд»ЈеҷЁд№ҹжҳҜеҗҢж ·зҡ„жҖқжғіпјҢдёҚйңҖиҰҒдёҖж¬ЎжҖ§е°ҶжүҖжңүз»“жһңи®Ўз®—еҮәжқҘдҝқеӯҳпјҢиҖҢжҳҜеҪ“йңҖиҰҒзҡ„ж—¶еҖҷиҝӣиЎҢи®Ўз®—е№¶з»ҷеҮәз»“жһңпјҢжңҖеӨ§зӣ®зҡ„е°ұжҳҜдёәдәҶиҠӮзңҒз©әй—ҙгҖӮ
д»Јз ҒзӨәдҫӢпјҡ

иҝҗиЎҢз»“жһңпјҡ

  иҝҷдёӘдё“йўҳиҝҳжІЎжңүи®ІеҲ°еҮҪж•°пјҢиҝҷйҮҢзЁҚеҫ®жңүзӮ№вҖңи¶…зәІвҖқж¶үеҸҠеҲ°еҮҪж•°дәҶгҖӮ
  д»ҘдёҠеңЁз”ҹжҲҗеҷЁеҮҪж•°йҮҢж”ҫзҪ®дәҶдёүдёӘyieldпјҢеҸҜд»ҘзҗҶи§ЈдёәдёүдёӘвҖңж–ӯз”өвҖқгҖӮз»ҸиҝҮеӨҡж¬Ўи°ғз”Ёе’Ңиҝ”еӣһеұҖйғЁеҸҳйҮҸе№¶жІЎжңүиў«йҮҠж”ҫгҖӮз”ҹжҲҗеҷЁеҮҪж•°IteratorFunctionе№¶дёҚиғҪзӣҙжҺҘз”ЁдәҺиҝӯд»ЈпјҢиҖҢжҳҜйҖҡиҝҮи°ғз”Ёе®ғеҗҺиҝ”еӣһиҝӯд»ЈеҷЁзҡ„зү№жҖ§пјҢеҲӣе»әдәҶTF1е’ҢTF2дёӨдёӘзӢ¬з«Ӣзҡ„иҝӯд»ЈеҷЁпјҢд№ҹеҚіиғҪз”ЁдәҺиҝӯд»Јзҡ„жҳҜTF1е’ҢTF2гҖӮеҸҚеӨҚи°ғз”ЁTF1е№¶дёҚдјҡж”№еҸҳTF2зҡ„иҝҗиЎҢдҝЎжҒҜгҖӮ
  ж”ҫзҪ®дәҶдёүдёӘyieldпјҢи°ғз”Ёдёүж¬ЎдјҡеҲҶеҲ«еҒңз•ҷеңЁдёүдёӘyieldдёҠпјҢеҰӮжһңи°ғз”Ёеӣӣж¬ЎеҸҠд»ҘдёҠдјҡеҸ‘з”ҹд»Җд№Ҳпјҹзӯ”жЎҲжҳҜдјҡжҠӣеҮәStopIterationејӮеёёпјҢеӣ дёәиҝӯд»ЈеҷЁе·Із»Ҹиҝӯд»ЈеҲ°иҫ№з•ҢдәҶпјҢд№ҹе°ұжҳҜжү§иЎҢеҲ°returnдәҶпјҢиҷҪз„¶жҲ‘д»¬зңҒз•ҘдәҶе®ғгҖӮyieldеҸӘжҳҜжҡӮеҒңreturnжҳҜзңҹжӯЈзҡ„з»“жқҹд№ҹж„Ҹе‘ізқҖиҝӯд»ЈеҲ°иҫҫиҫ№з•ҢгҖӮдёҖиҲ¬жҲ‘д»¬дјҡеңЁз”ҹжҲҗеҷЁеҮҪж•°йҮҢж”ҫдёҠеҫӘзҺҜжқҘдҪҝз”ЁгҖӮ
  дёӢйқўиҝҷдёӘдҫӢеӯҗз”Ёз”ҹжҲҗеҷЁе®һзҺ°дәҶж–җжіўйӮЈеҘ‘ж•°еҲ—зҡ„иҝӯд»ЈгҖӮ

иҝҗз®—з»“жһңпјҡ

  д»ҘдёҠдҫӢеӯҗдёәдәҶз®ҖеҚ•еҢ–е’ҢиҜҙжҳҺдҪңз”ЁпјҢжҲ‘е°ҶwhileеҫӘзҺҜеҶҷжҲҗдәҶжӯ»еҫӘзҺҜпјҢд№ҹеҚіиҝҷдёӘиҝӯд»ЈеҷЁжҳҜжІЎжңүз»“жқҹвҖқиҫ№з•ҢвҖңзҡ„пјҢзҺ°е®һи®Ўз®—жңәдёҚиғҪж”ҜжҢҒж— йҷҗж¬Ўи°ғз”Ёзҡ„иҝҗз®—йҮҸпјҢиҝҷжҳҜдёҚеҘҪзҡ„зӨәиҢғпјҢжҳҜдёәдәҶжҸҗйҶ’еӨ§е®¶дёҚиҰҒеҸӘе…іжіЁиҝӯд»ЈеҷЁиҝӯд»ЈжҺЁеҜјзҡ„и§„еҲҷпјҢиҝҳйңҖиҰҒе…іжіЁиҫ№з•ҢгҖӮ
йҮҚжһ„иҝҷж®өд»Јз Ғпјҡ

иҝҗиЎҢз»“жһңпјҡ

  еҸҜд»ҘзңӢеҲ°жҲ‘д»¬йҷҗе®ҡдәҶиҝӯд»Јж¬Ўж•°дёә5ж¬ЎпјҢеҪ“з¬¬е…ӯж¬Ўиҝӯд»Јж—¶з”ұдәҺжү§иЎҢеҲ°returnиҝӯд»ЈеҲ°иҫҫиҫ№з•ҢеҒңжӯўпјҢжҠӣеҮәдәҶејӮеёёStopIteration: -1пјҢе…¶дёӯ-1ж—ўжҳҜжҲ‘д»¬returnзҡ„еҖјгҖӮ
