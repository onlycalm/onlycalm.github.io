cиҝӣйҳ¶зҜҮпјҲдә”пјүвҖ”вҖ”еҢҝеҗҚз»“жһ„дҪ“

еүҚиЁҖпјҡ
&emsp;&emsp;еҜ№c/c++зЁӢеәҸе‘ҳжқҘиҜҙпјҢз»“жһ„дҪ“жҳҜйқһеёёеёёз”Ёзҡ„иҮӘе®ҡд№үж•°жҚ®зұ»еһӢпјҢе®ғеҜ№ж•°жҚ®иҝӣиЎҢе°ҒиЈ…пјҢдҪҝз”ЁеҢҝеҗҚз»“жһ„дҪ“еҸҜиҝӣдёҖжӯҘеўһејәж•°жҚ®зҡ„е°ҒиЈ…жҖ§гҖӮ

1 д»Җд№ҲжҳҜеҢҝеҗҚз»“жһ„дҪ“

&emsp;&emsp;д»ҘдёӢжҳҜжӯЈеёёе®ҡд№үдёҖдёӘеёҰеҗҚз§°зҡ„з»“жһ„дҪ“зӨәиҢғгҖӮ

&emsp;&emsp;еҗҚдёә STRect зҡ„з»“жһ„дҪ“жӢҘжңүдёӨдёӘжҲҗе‘ҳ byLen е’Ң byWide пјҢиҰҒеҜ№е…¶и®ҝй—®еҸҜд»ҘйҖҡиҝҮдёҖдёӘжҲҗе‘ҳиҝҗз®—з¬ҰеҚіеҸҜи®ҝй—® stRect.byLen жҲ– stRect.byWide гҖӮ
&emsp;&emsp;еҰӮжһңжҲҗе‘ҳеӨҡеҮ дёӘ зұ»еҲ« пјҢеҫҖеҫҖдјҡиҝӣиЎҢдәҢж¬Ўе°ҒиЈ…пјҢжҜ”еҰӮзҹ©еҪўжңүе…ідәҺе°әеҜёзҡ„жҲҗе‘ҳеҸҳйҮҸпјҢд№ҹжңүе…ідәҺйўңиүІзҡ„жҲҗе‘ҳеҸҳйҮҸгҖӮ

&emsp;&emsp;йҡҸзқҖе°ҒиЈ…зҡ„еұӮзә§и¶ҠеӨҡпјҢи®ҝй—®ж—¶еҶҷеҮәзҡ„иЎЁиҫҫејҸеҸҲй•ҝеҸҲйҡҫзңӢпјҢдёәдәҶдҝқиҜҒе°ҒиЈ…жҖ§зҡ„еҗҢж—¶еҸҲж–№дҫҝеҸҳйҮҸи°ғз”ЁпјҢеҸҜд»ҘйҮҮз”ЁеҢҝеҗҚз»“жһ„дҪ“гҖӮ

2 з»“жһ„дҪ“еөҢеҘ—еҢҝеҗҚз»“жһ„дҪ“

&emsp;&emsp;еҢҝеҗҚз»“жһ„дҪ“жІЎжңүз»“жһ„дҪ“еҗҚпјҢеҸҜд»ҘзӣҙжҺҘи®ҝй—®еҢҝеҗҚз»“жһ„дҪ“зҡ„жҲҗе‘ҳгҖӮеҢҝеҗҚз»“жһ„дҪ“йңҖиҰҒеөҢеҘ—еңЁе…·еҗҚз»“жһ„дҪ“жҲ–иҒ”еҗҲеҶ…пјҢзӣҙжҺҘе®ҡд№үж— еҗҚз»“жһ„дҪ“е°Ҷж— жі•иҝӣиЎҢи®ҝй—®пјҢиҜӯжі•дёҚе…Ғи®ёд№ҹж— ж„Ҹд№үгҖӮдёҠйқўзҡ„з»“жһ„дҪ“жҲ‘д»¬еҸҜд»ҘиҝӣиЎҢз®ҖеҢ–дёәд»ҘдёӢе®ҡд№үгҖӮ

3 иҒ”еҗҲеөҢеҘ—еҢҝеҗҚз»“жһ„дҪ“

&emsp;&emsp;еҢҝеҗҚз»“жһ„дҪ“жӣҙеёёз”ЁдәҺиҒ”еҗҲдёӯпјҢйҖҡиҝҮеөҢеҘ—еңЁиҒ”еҗҲдёӯеҚіеҠ ејәзҡ„дәҶж•°жҚ®зҡ„е°ҒиЈ…жҖ§пјҢд№ҹжҳҜзҡ„еҜ№ж•ҙдҪ“е’ҢеұҖйғЁж•°жҚ®зҡ„иөӢеҖје’Ңи®ҝй—®ж“ҚдҪңжӣҙж–№дҫҝгҖӮ
