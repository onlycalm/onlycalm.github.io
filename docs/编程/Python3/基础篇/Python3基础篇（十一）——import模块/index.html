阅读这篇文章我能学到什么？
&emsp;&emsp;import关键字用于导入现有的模块，增强了代码的重用性。可以和from和as关键字一起使用。这篇文章将为你介绍import的几种用法。

1 import [ModularName]

1.1 导入一个模块

&emsp;&emsp;为了实现代码的重用性，方便将来或他人使用，我们通常会将代码进行模块化。而Python3中提供了import关键字导入现成的模块，导入的模块即可以是开发者自己实现的，也可以是第三方库。python提供了丰富的第三方库，使得大多数功能都不需要自己去重复实现。
