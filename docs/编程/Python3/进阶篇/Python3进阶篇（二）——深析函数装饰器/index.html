Python3进阶篇（二）——深析函数装饰器

前言：
阅读这篇文章我能学到什么？
&emsp;&emsp;装饰器可以算Python3中一个较难理解的概念了，这篇文章由浅入深带你理解函数装饰器，请阅读它。

1 装饰器基本概念

&emsp;&emsp;装饰器是能够修改已定义函数功能的函数，也即装饰器本身就是具有这种特殊功能的 函数 。修改是有限制的修改，它只能在你定义的函数执行前或执行后执行其他代码，当然也能给你的函数传递参数。不能直接修改你定义的函数内部的代码。
&emsp;&emsp;举个通俗的例子。比如你定义了函数A，那么函数A被装饰器装饰之后，此时你调用A，它可能先去执行一些动作（代码）然后执行函数A，完了又去执行另外一些动作（代码）。也即装饰器帮你的函数附加了一些动作（代码）执行，它改变了你原来定义的函数功能。
&emsp;&emsp;如果你看过我的上一篇进阶篇关于函数的讲解（其中降到了函数嵌套定义、函数作为参数、函数返回函数等问题），那么后续的内容将会更容易理解。
