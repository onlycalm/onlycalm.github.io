英语基础语法（五）——副词

前言：
  副词在句中主要做状语，也有比较级和最高级的变化。

1 什么是副词

  用来修饰动词、形容词、其他副词，表示时间、地点、方式、程度等的词称为副词。副词主要用来做状语，用来表示行为或状态特征，有比较级和最高级变化。副词按照语法作用可以分为四类：疑问副词、连接副词、关系副词、普通副词。

2 副词的分类

  • 疑问副词：用于引出特殊疑问句的when, where, why, how等。
  • 链接副词：这类副词有相当于并列连词的therefore, besides, otherwise, however, moreover, furthermore, still, thus, meanwhile等，也有引导名词性从句或不定式的when, why, where, how等。
  • 关系副词：用于引导定语从句并在句中充当状语的when, where, why等。
  • 普通副词：除上述三类副词之外的副词都属于普通副词，普通副词按其词义和用法又可分为四类：
种类 用法 举例
时间副词 表示时间 now, today, yet, already, before, after, then等
地点副词 表示地点或位置关系 here, there, out, back, off, below, inside, down等
方式副词 表示行为动作发生的方式 well, fast, slowly, happily, badly, together等
程度副词 表示程度 very, too, quite, almost, enough, greatly, still等

3 副词的构词

  常见的副词构词方式大致有三种：形容词 + ly、-ward/wards 和 -wise。除此之外，有些词本身就可充当副词，有些副词与形容词或介词同形，而有些副词兼有两种形式。
  有些以-ly结尾的词并非副词，而是形容词。

3.1 形容词 + ly

  大多数副词都是由“形容词 + -ly”构成的。“形容词 + -ly”变成副词主要有以下几种情况：

  • 一般情况下，直接在形容词词尾加 -ly。
  • 以辅音字母加y结尾的形容词要变y为i，再加-ly。
  • 以辅音 + le结尾的词要去e加y。
  • 少数以元音 + e结尾的形容词，去掉e，再加-ly。

3.2 -ward/wards

  有些副词是由介词或地点名词加后缀-ward(s)构成的，意为“向…”，“朝…”。

词组 含义
downward(s) 向下
backward(s) 向后
seaward(s) 向海
upward(s) 向上
inward(s) 向内
homeward(s) 向家地
sunward(s) 向阳
northward(s) 向北
outward(s) 向外

3.3 -wise

  有些副词是由名词加后缀-wise构成的，表示位置、方向、状态等相关意义。

词组 含义
crosswise 交叉地
coastwise 沿海岸
cornerwise 对角地
lengthwise 纵长地
endwise 末端朝前地
sidewise 斜向一边地
likewise 同样地
otherwise 否则
pairwise 成双成对地

4 其他特殊形式的副词

  • 简单副词：指本身可以充当副词的词，如：
词组 含义
here 在这里
there 在那里
often 常常
always 一直，总是
  • 与形容词同形的副词。列如：daily, early, fast, hard, high, long, near, straight, well等。请比它们的词义和用法：
词组 形容词 副词
high the high jump 跳高项目 to jump high 跳得高
fast a fast car 开得快的汽车 to drive fast 开得快
early an early riser 早起的人 to get up early 起得早
straight a straight line 直线 go straight ahead 一直朝前走

  这两类词含义基本相同，区别在于它们的语法作用不同，修饰名词或做表语的是形容词；修饰其他词类、短语或从句的是副词。

  • 与介词同形的副词
      英语中还有一些与介词同形的副词，大多是关于动作、方向、位置和地点等方面的，列如：above, over, down, up, in, on, off, at, around, along, near, outside等，其主要区别是介词后接宾语，而副词后面不接宾语。
  • 兼有两种形式的同根副词
      有些副词具有两种形式，一种与形容词同形，一种是由形容词加后缀-ly构成的，如：late, lately; short, shortly; wide, widely; clear, clearly; hard, hardly等。这两种形式的副词有时含义相同，有时不同，其用法也不尽相同。

5 副词的功能

  副词可以做状语、定语、表语、宾语、宾语补足语等。

  • 做状语：用来修饰动词、形容词、其他副词或整个句子
  • 做定语：少数表示时间或地点的副词可做定语
  • 做表语：某些表示位置的副词以及某些与介词同形的副词可以用做表语。
  • 做宾语。
  • 做宾语补足语。

  副词用做宾语的用法十分有限。通常只用做介词，并且只限于某些表示时间和地点的副词。而且不同的副词有不同的搭配特点，如here和there可与along, around, down, from, in, near, round, up等介词连用，但通常不与介词to连用（from here to there 是例外）；而表示地点的副词abroad则只与介词from连用，不与其他介词连用。

  • 做宾语补足语。一般来说能用做表语的副词都可用做宾语补足语。

6 副词的位置

  副词是英语中用法比较灵活的一类词，因此其在句子中的位置也并不总是固定的。

6.1 一般情况下

  • 副词一般位于be动词、助动词或情态动词之后，实义动词之前。如果实义动词后有宾语，则放于宾语之后。
  • 副词在修饰形容词、副词时，一般放在被修饰词之前，但enough除外。
  • 疑问副词、连接副词、关系副词以及修饰整个句子的副词，通常放在句子或从句的前面。

6.2 多个副词连用时

  • ;当时间、地点、程度与方式副词同时在句中出现时，其顺序一般为：程度副词 + 方式副词 + 地点副词 + 时间副词（时间副词也可置于句首）。但是，当句首中动词是动态动词（go, come, leave, arrive, etc.）时，则常把地点副词放在方式副词之前。
  • 当程度副词与频度副词同时在句中出现时，程度副词通常前置修饰频度副词。否定副词放在句首时，句子要部分倒装。副词only的位置比较灵活，修饰哪个词，就放在那个词的前面表示强调。

7 副词的比较等级

7.1 副词比较等级的构成

  副词和形容词相同，在进行比较时，有原级、比较级和最高级三种等级。其比较级和最高级的变化与形容词相似。

  • 规则变化
构词方法 原级 比较级 最高级
一般在词尾加-er或-est fast faster fastest
以字母e结尾的词，加-r或-st late later latest
以“辅音字母+y”结尾的单次，将y改为i再加-er或-est heavy heavier heaviest
多音节词在词前加more或most easily more easily most easily
  • 不规则变化

| 原级 | 比较级 | 最高级 |
| well | better | best |
| little | less | least |
| much | more | most |
| badly | worse | worst |
| far | farther（表距离） | farthest |
| far | further（表程度） | furthest |

7.2 副词比较等级的用法

  副词比较等级的用法也与形容词比较等级的用法相似。

  • 两者比较（比较级）：两个人或事物进行比较。
    • A = B: A + 谓语 + as + 副词原级 + as + B。
    • A ≠ B: A + 谓语 + not + as/so + 副词原级 + as + B。
    • A >/< B: A + 谓语 + 副词比较级 + than + B。
  • 三者及以上比较（最高级）：三个或三个以上的人或事物进行比较。A > B, C, D: 主语 + 谓语（+ the） + 谓语最高级 + 比较对象。
  • 副词比较等级的特殊作用。
    • “比较级 + and + 比较级”，表示“越来越…”。
    • “the + 比较级，the + 比较级”，表示“越…就越…”。
    • “no + 副词比较级 + than”，表示对两个比较对象都进行否定（可以用neither…nor…结构来改写）。
    • 副词的比较级形式还可表示最高级含义。”比较级 + than + any(other) + 单数可数名词“或”比较级 + than + any of the other/all the other + 复数名词“表示”甲比同一范围的任何一个人/物都…“，含义是”甲最…“。

8 副词的用法辨析

  有几组副词意思相近，容易混淆。

8.1 already/still/yet用法辨析

  • already/yet
    • 两者都有“已经”、“至今”的意思，并常用于完成时态中。但already一般用于肯定句中，而yet一般用于否定句和疑问句中。
    • already也可用于疑问句中，一般放在句尾，表示动作比预期的发生得早，带有“惊讶”的意思；而yet用于疑问句中则表示动作比预期的发生得晚。
      still/yet。两者都有“仍然”、“还”的意思。但still常用在肯定句中，表示“某事仍在进行中”；而yet一般用在否定句中，表示“某事至今还未发生”。

8.2 too/either/alse用法辨析

  • too/either。两者都有“也”的意思，并都用在句尾。但too用在肯定句中，either用在否定句中。
  • also。also也有“也”的意思，但also常用在肯定句或疑问句中，且一般放在be动词之后，其他动词之前。

8.3 so/neither(nor)用法辨析

  so和neither(nor)两词都可用于倒装句的开头。so用于肯定句，表示“也”；neither(nor)用于否定句，表示“也不”。

8.4 ago/before用法辨析

  • ago表示”从现在来看的以前“，常与一般过去时和过去进行时连用。
  • before表示”从过去某个时间来看的以前“，常与一般过去时、过去完成时连用。
    • before与过去完成时连用时，表示“before then”之意，即“过去某一时间以前”的意思。 * before与一般过去时和现在完成时连用时，表示“before new, still now”之意，即“现在以前，迄今为止”的意思。
  • 表示“从前”之意时，只能用before，而不能用ago。

8.5 ever/never用法辨析

  • never
    • never主要与肯定动词连用而不与否定动词连用。一般情况下“从未有过”。
    • “never + 肯定动词”有时可以代替一个普通的否定动词。
    • “never + 疑问动词”可以表示说话人对于未能做到某事而感到惊讶。
  • ever
    • ever意为“在任何时间”，主要用于疑问句中。
    • ever可与否定动词连用，特别是在与复合时态连用时，可代替“never + 肯定动词”。
    • “ever + 肯定动词”可以用于比较，也可以和表示嘉定和怀疑的词连用。

8.6 sometimes/sometime/some time/some times用法辨析

  • sometimes是个频度副词，意为“有时候”，多用于一般现在时，对它进行提问常用how often。
  • sometime也是一个副词，意为“在某一时候（过去或将来）”，表示的是任何时候或某一不确定的时间，对它提问用when。
  • some time是名词短语，意为“一段时间”，其中time是不可数名词，对它进行提问要用how long。
  • some times是名词短语，意为“几次（倍）”，这里的time是可数名词，对它进行提问要用how many times。
